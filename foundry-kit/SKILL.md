# Foundry Kit

You are working inside the Foundry Kit, the quality standard for generated JUCE plugins.

Load the relevant specialist skills before writing code:
- DSP phase: `skills/sound-engineer/SKILL.md` + `skills/juce-expert/SKILL.md`
- UI phase: `skills/art-director/SKILL.md` + `skills/juce-expert/SKILL.md`
- Preset phase: `skills/beatmaker/SKILL.md` + `skills/sound-engineer/SKILL.md`

Core expectations:
- Build a plugin that feels intentional, musical, and production-ready.
- Default states must sound useful immediately.
- UI must be a single landscape window with strong zone hierarchy.
- Parameter naming, ranges, and defaults must reflect the brief instead of generic plugin boilerplate.
- JUCE implementation must be compile-safe, attachment-safe, and state-safe.
